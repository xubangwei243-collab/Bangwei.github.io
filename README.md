# Bangwei Xu Portfolio – Terminal‑Chic Data Visual Style

This redesigned site embraces a dark, data‑driven aesthetic that echoes high‑performance quant systems. A subtle interactive point‑cloud background evokes geospatial and multi‑agent dynamics while keeping focus on the content. A scrolling ticker bar communicates technical expertise continuously. Glass‑like cards with blur and depth add modern polish, and micro‑charts animate key metrics. Hover glows and a neon accent palette complete the high‑end feel.

## Key Features

- **Interactive Hero:** A low‑opacity network animation responds to cursor movement, representing complex data systems without distracting from your message. Built with Vanta.js.
- **Live Ticker Bar:** A scrolling ticker at the top cycles through your technical toolkit and accomplishments, giving the site a sense of continuous analysis.
- **Glassmorphism:** Content cards use translucent dark backgrounds, backdrop blur and neon glows to create depth and sophistication. Hover states lift cards slightly and intensify the glow.
- **Micro‑Charts:** Small charts and animated pulses replace static icons: a circular progress shows your perfect GPA, a donut indicates false‑alarm reduction and a pulsing dot conveys high‑frequency data flow.
- **Dark Theme:** A rich black background with deep grey panels and cyan accents (#00d4ff) lets data visualisations pop, while optional red tones echo Cornell’s branding.

## Running the Site

Open `index.html` directly in your browser or serve the folder locally:

```bash
cd bangwei-website
python3 -m http.server --bind 127.0.0.1 8000
```

Then navigate to `http://localhost:8000` in your browser. The site is static and deployable on GitHub Pages, Netlify or Vercel.
